# 矩阵

## 矩阵的加法

A 是 m × n 的矩阵，B 是 m x n 的矩阵，那么它们的乘积 C = A + B 是 m × n 的矩阵，其中 C 的第 i 行第 j 列的元素是 A 的第 i 行和 B 的第 j 列的和

## 矩阵乘法

A 是 m × n 的矩阵，B 是 n × p 的矩阵，那么它们的乘积 C = A × B 是 m × p 的矩阵，其中 C 的第 i 行第 j 列的元素是 A 的第 i 行和 B 的第 j 列的点积
