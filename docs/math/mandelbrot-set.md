# 曼德博集合

曼德博集合(Mandelbrot Set) 是指一种在复平面上组成分形的点的集合，以数学家本华·曼德博的名字命名

## 定义

曼德博集合可以用复二次多项式来定义：$f_c{z} = z^2 + c$, $c$ 是一个复数参数

从 $ z = 0 $ 开始，迭代计算：
    $ z_{n+1} = z_n^2 + c $
    $ z_{0} = 0 $
    $ z_{1} = z_0^2 + c = c $
    $ z_{2} = z_1^2 + c = c^2 + c $

不同的参数 $c$ 可能使序列的绝对值逐渐发散到无限大，也可能收敛在有限的区域内
曼德博集合就是使序列不延伸至无限大的所有复数 $c$ 的集合
