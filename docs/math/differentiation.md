# 数值微分

## 两点法

前向微分
f'(x)h = f(x + h) - f(x)

后向微分
f'(x)h = f(x) - f(x - h)

## 三点法

前向微分
f'(x)* 2h = -3f(x) + 4f(x + h) - f(x + 2h)

后向微分
f'(x)* 2h = f(x - 2h) - 4f(x - h) + 3f(x)

中心公式
f'(x)* 2h = f(x + h) - f(x - h)

## 五点公式
f'(x) *12h = f(x - 2h) - 8f(x - h) + 8f(x + h) - f(x + 2h)

## 二阶导
f''(x) h^2 = f(x + h) - 2f(x) + f(x - h)
